constexpr IE::Core::Logger::Flags IE::Core::Flags ( uint8_t  t_flags)
constexpr

Basic private 'constructor' for the Flags enum.

This is used to create IE::Graphics::Image::Locations from the uint8_ts that are used to perform the binary operators. Before converting the t_flags to an IE::Core::Logger::Flags, validity checks are performed to ensure that the t_flags does not represent a physically impossible scenario.

Parameters
t_flagsThe value that the new Location should take on.
Returns
A verified IE::Graphics::Image::Location.
